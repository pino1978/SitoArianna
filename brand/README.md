# Brand Assets - Arianna Milano

Questa cartella contiene gli asset grafici del brand Arianna Milano.

## Fonti

`../brand-kit.md` e la fonte autorevole per identita, posizionamento, tono, direzione visiva e qualita attesa.

`brand-generation-guide.md` e la guida operativa per generare, salvare, revisionare e promuovere asset.

`lettering/lettering-system.md` e la fonte autorevole per il sistema lettering.

## Regole Di Uso

Gli asset approvati stanno nelle cartelle definitive:

- `logo/`
- `lettering/`
- `textures/`
- `icons/`
- `images/`

`generated/` contiene solo bozze, asset temporanei o materiali non ancora approvati.

Non creare metadata JSON per ogni asset statico. La categoria e indicata dalla cartella, mentre le regole di brand stanno nel Brand Kit e nei documenti di sistema.

Ogni nuovo asset deve essere una estensione coerente di un sistema grafico esistente. Se il sistema non esiste, va progettato e documentato prima di generare l'asset finale.
